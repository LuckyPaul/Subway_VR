Holo NGUI is a beautiful Sci-Fi GUI skin for Unity3D.

Holo NGUI is a beautiful Sci-Fi theme for NGUI.
Requires NGUI 3.5.0 or later!

All the fonts and textures are in a single atlas.

The package contains:
- Demo scene.
- A window class which allows you to focus and fade in and out windows.
- 21 Unique Elements
- 11 Icons
- PSD file.
